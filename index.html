How to access your mods folder

Remember, you will need to download and install CurseForge to access the main modpack. The standalone version is recommended.

After installing CurseForge and the Dungeon Heroes (RPG Series) modpack, continue to the following steps:

  1. Open the CurseForge client and navigate to "My Modpacks"" in the Minecraft tab.
  2. Run Dungeon Heroes for the first time to let the files populate onto your PC. This may take some time.
  3. Once you've reached the Minecraft main menu and everything is working, close the game.
  4. Go back to the CurseForge client.
  5. Right click on Dungeon Heroes and select "Open Folder" to open the modpack's file location on your PC. The directory should look something like this: "C:\Users\[username]\curseforge\minecraft\Instances\Dungeon Heroes (RPG Series)"
  6. Navigate to your mods folder here. If you would like to add any additional resource packs, you would also do that in this directory.
